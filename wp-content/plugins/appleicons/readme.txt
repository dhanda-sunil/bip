== Description ==

Simply add the Icons and loading screens for various versions of iPhone and iPad.

== Installation ==

Upload the Appleicons plugin to your plugin directory, Activate it, then upload the images which you want to show on iphone during loading screen.

1, 2, 3: You're done!
